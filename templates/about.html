About SimpleApp

SimpleApp is a beginner-friendly Django project designed to demonstrate core features of web applications like:

  • User authentication (sign up, log in, log out)
  • Basic CRUD (Create, Read, Update, Delete) for personal notes
  • Bootstrap-based UI for clean and responsive design

This project is ideal for learning Django basics without using Django REST Framework.
