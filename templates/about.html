About NoteDesk

NoteDesk is a personal note management web application built using Django. It allows users to create, save, and manage their own notes securely.

Each user has their own private space where they can:

  • Create Notes: Add a note with a title and detailed description.
  • View Notes: Access only your own saved notes after logging in.
  • Edit & Delete: Update or remove your notes as needed.
  • Authentication: Secure user login, signup, and logout system.
  • Responsive Design: Clean and mobile-friendly interface using Bootstrap.

NoteDesk is ideal for anyone who wants a simple, secure, and personalized note-taking experience while also learning core Django concepts like models, views, templates, and user authentication.
